# aquagesti
B2B platform for managing water treatment installations, maintenance visits and customers.
The project is built using .NET and follows a Modular Monolith architecture with Domain Driven Design (DDD) principles.

## Tech Stack
- .NET 8
- Blazor
- Modular Monolith Architecture
- Domain Driven Design (DDD)
- PostgreSQL
- Docker
- GitHub Actions

## Functional Area
- Customer Management
- Installation Management
- Visit Scheduling
- User Access
